## Project description
Shipments CRUD page

This project is an assesment for Kuhne/Nagel front-end development internship. This project taught me how to integrate a CSS template into React project, Redux(including Thunk middleware for making asynchronous actions) and Axios.

### Features
- View shipment data in a table
- Delete shipment
- View and edit a shipment

## Run the project
You need to have [NodeJS](https://nodejs.org/en/) installed.
1. Install neccesary dependencies `npm install --legacy-peer-deps`
2. In the project root directory, start the program `npm start`
3. Open [http://localhost:3000](http://localhost:3000) to view the project in your browser

## Stack
- React
- Redux
- Axios
- Black Dashboard React(CSS template)
